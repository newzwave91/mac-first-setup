🍎 맥 처음 세팅 도우미

아래 3단계만 따라 하면, 나머지는 화면에 뜨는 창에서 클릭만 하면 됩니다.

1터미널 열기

키보드에서 ⌘ command + 스페이스바를 함께 누르고, 터미널이라고 입력한 뒤 return을 누르세요.
까만(또는 하얀) 글자 입력 창이 하나 열립니다. 이게 터미널이에요.

2아래 명령 복사하기

bash -c "$(curl -fsSL RAW_URL_PLACEHOLDER)"

3터미널에 붙여넣고 return

터미널 창을 클릭한 뒤 ⌘ command + V로 붙여넣고 return을 누르세요. 잠시 후 선택 창이 뜨면 이제 클릭만 하면 됩니다.

🔑 중간에 암호를 물어보면 맥에 로그인할 때 쓰는 암호를 입력하세요. 입력해도 화면에 아무것도 표시되지 않지만 정상적으로 입력되고 있는 거예요.
