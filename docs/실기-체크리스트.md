# 실기 테스트 체크리스트 (Task 10 Step 4 — 사용자 수행)

리뷰 게이트에서 이월된 항목(표기)을 반드시 포함해 주세요.

## 준비

- [ ] `MFS_DRY_RUN=1 bash setup.sh` 로 GUI 다이얼로그 흐름 먼저 눈으로 확인 (실변경 없음)
- [ ] 실제 실행 전 현재 설정 상태 캡처(트랙패드·Finder 설정 화면 스크린샷 권장)

## 다이얼로그 흐름

- [ ] 환영 창 → 프로필 → 앱 선택(추천 미리 체크됨) → 설정 선택 → 완료까지 클릭만으로 진행되는지
- [ ] **[리뷰 이월] 각 단계에서 [취소]를 눌러 안전 종료되는지 — 환영/프로필/앱/설정 4곳 각각 1회** (Task 4 리뷰: 취소 경로는 자동 테스트 불가)
- [ ] 완료 창의 [리포트 열기]가 텍스트 편집기로 리포트를 여는지

## 설치 경로

- [ ] 미설치 앱 1개 이상이 실제로 설치되는지 (설치됨 항목은 "건너뜁니다" 로그)
- [ ] 재실행 시 이미 설치된 항목이 전부 스킵되는지 (멱등성)
- [ ] **[리뷰 이월 I-3] 카카오톡(mas) 경로 실측 — App Store 로그인 상태와 미로그인 상태 각각: 미로그인이면 실패 + "직접 설치" 수동 안내가 리포트에 남는지**

## 설정 실효성 — **[리뷰 이월 I-2] 8종 각각 실제 적용 확인** (로그아웃 후 재로그인하고 판정)
